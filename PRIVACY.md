\# Privacy Policy



Pixiv Bookmark Random does not collect, store, or transmit any personal data.



\## Data Collection

The extension does not collect personally identifiable information, browsing history, user activity, or website content. All operations are performed locally within the user's browser.



\## Data Usage

The extension accesses Pixiv bookmark pages only to retrieve artwork links for the purpose of randomly opening a bookmarked illustration. No data is stored, analyzed, shared, or transmitted to any external server.



\## Third-Party Services

The extension does not use analytics services, advertising networks, or third-party trackers.



\## Remote Resources

If remote resources are used, they are limited to static, non-executable configuration data. No remote JavaScript code is executed.



\## Data Sharing

No user data is sold, transferred, or shared with third parties.
